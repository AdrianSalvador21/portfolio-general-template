    Buttons

    All buttons have been based upon the Bootstrap buttons.

    Types of Buttons

    There are three buttons that you can use:

    • btn-primary - Main button
    • btn-outline-primary - Ghost button
    • btn-outline-light - Ghost button with white text color

    You can add the buttons as follow:

    <button type="button" class="btn btn-primary">Click here</button>
<button type="button" class="btn btn-outline-primary">Click here</button>
<button type="button" class="btn btn-outline-light">Click here</button>

    Add icons

    It's also possible to add an icon inside the button. Just find the icon you want to use and add the HTML code inside of it:

    <button type="button" class="btn btn-primary"><i class="lni-coffee-cup"></i> Click here</button>

    Extra classes

    You can also give a flat shadow to the button and/or make the button wider with btn--shadow btn--wider:

    <button type="button" class="btn btn-primary btn--shadow btn--wider">Click here</button>

    Scroll to a div in page function

    When a user clicks on a button you can make the user scroll to a div ID inside the page by adding the id as an href with an hash before it. Don't forget to add the smoothscroll class, otherwise the button won't scroll the page to the div.

    <!-- Button with contact as an id, don't forget the hash -->
<button type="button" href="#contact" class="btn btn-primary smoothscroll">Hire Me</button>

<!-- Somewhere on the page the id is contact -->
<section class="contact" id="contact">
