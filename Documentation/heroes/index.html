    Heroes

    There are 4 heroes for the template:

    • Default (Static image)
    • Carousel
    • Video
    • Parallax

    Default

    The hero is pretty straightforward if you've read this documentation. You can change the image of the hero by changing the url of the image. A linear-gradient has been added to darken the image by 20%. You can remove or edit this as you want. Inside the hero__text div you can edit all the text and the button. At the end you can find the scrolling mouse icon. You can remove it if you don't want to use it. 

    <section class="hero mb-4 mb-md-5">
    <div class="hero__wrapper d-flex align-items-center aos-init aos-animate" style="background-image:  linear-gradient(rgba(0, 0, 0, 0.2), rgba(0, 0, 0, 0.2)), url('/assets/img/hero/hero-pexels-photo-840666.jpg')" data-aos="fade" data-aos-delay="800" data-aos-duration="1000">
        <div class="container">
            <div class="hero__text">
                <div class="subtitle aos-init aos-animate" data-aos="fade-right" data-aos-delay="800" data-aos-duration="900">
                    Nice to meet you
                </div>
                <h1 data-aos="fade" data-aos-delay="1400" data-aos-duration="1000" class="aos-init aos-animate">Anthony Douglas</h1>
                <p class="lead aos-init aos-animate" data-aos="fade-right" data-aos-delay="1800" data-aos-duration="900">I'm a <span>Creative Developer</span></p>
                <button type="button" href="#contact" class="btn btn-outline-light btn--wider mt-4 smoothscroll aos-init aos-animate" data-aos="fade" data-aos-delay="2600" data-aos-duration="1000">Hire Me</button>
            </div>
            <div class="ui-scroll" id="ui-scroll">
                <small>Scroll</small>
                <span class="scroll-icon">
                    <span class="scroll-icon__wheel-outer">
                        <span class="scroll-icon__wheel-inner">&nbsp;</span>
                    </span>
                </span>
            </div>
        </div>
    </div>
</section>

    Video

    Video's can be extremely big. Please try compress them as much as possible as it slows down your page a lot. The video can be swapped as follow:

    <video autoplay muted loop id="heroVideo" class="hero__video">
    <source src="assets/video/video.mp4" type="video/mp4">
</video>

    Parallax

    This hero uses Jarallax for the parallax effect. Please check out there documentation for more information

    <section class="hero mb-4 mb-md-5">
    <div class="hero__wrapper d-flex align-items-center">
        <div data-jarallax data-speed="0.2" class="jarallax hero__parallax">
            <img src="{{{hero-img}}}" alt="Jarallax" data-speed="0.7" class="jarallax-img jarallax-img--darken">
            <div class="jarallax__content d-flex justify-content-center align-items-center" data-aos="fade" data-aos-delay="300" data-aos-duration="1000">
                <div class="container ">
                    <div class="hero__text">
                        <!-- Content -->
                    </div>
                    <div class="ui-scroll" id="ui-scroll">
                        <small>Scroll</small>

                            </span>
