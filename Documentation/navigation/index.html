    Navigation

    The navigation has mainly been created for an onepage interaction. It does work as well for multipage interactions. The navigation has been built upon the Bootstrap navbar. This is the main structure:

    <nav class="navbar navbar-expand-lg fixed-top addscrollspy" id="navbar">
    <a class="navbar-brand font-weight-bolder mr-xl-3" href="#">
        <img class="navbar-logo navbar-logo--inverted" src="/assets/img/jackdaw-logo-inverted.svg" alt="Jackdaw">
        <img class="navbar-logo navbar-logo--default" src="/assets/img/jackdaw-logo-default.svg" alt="Jackdaw">
    </a>
    <button class="navbar-light navbar-toggler" type="button" data-toggle="collapse" data-target="#navbars-default" aria-controls="navbars-default" aria-expanded="false" aria-label="Toggle navigation">
        <span class="navbar-toggler-icon">
        <span class="burger">
            <span class="burger__line"></span>
            <span class="burger__line"></span>
            <span class="burger__line"></span>
        </span>
        </span>
    </button>
    <div class="collapse navbar-collapse ml-auto mr-auto" id="navbars-default">
        <ul class="navbar-nav ml-auto mr-auto">
            <li class="nav-item">
                <a class="nav-link active" href="#index">Home</a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#about">About</a>
            </li>
            <li class="nav-item">

        <span class="navbar__slideline" id="slideline"></span>
    </div>
</nav>

    Logo

    You can replace the logo in the navbar-brand div. It's possible to use one logo as well. Just remove one of the two logo and the extra class like: navbar-logo--inverted, or navbar-logo--default. Please check out the index-dark.html as an example for one logo only. This is an example for the logo swap version:

    <a class="navbar-brand font-weight-bolder mr-xl-3" href="#">
    <img class="navbar-logo navbar-logo--inverted" src="/assets/img/jackdaw-logo-inverted.svg" alt="Jackdaw">
    <img class="navbar-logo navbar-logo--default" src="/assets/img/jackdaw-logo-default.svg" alt="Jackdaw">
</a>

    Navigation Items

    You can easily link the navigation items to a div on the page. Just make sure that the div on the page has got the same id as the href. Example:

    <!-- Navigation item with about as an id, don't forget the hash -->
<a class="nav-link" href="#about">About</a>

<!-- Somewhere on the page the id is about -->
<section class="about" id="about">

    You can also link to other pages. Just remove the hash in the href and the JS script won't be looking for a div ID in the page any longer.
