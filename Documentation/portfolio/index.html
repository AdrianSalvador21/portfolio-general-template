    Portfolio

    The portfolio has been built with Isotope. It has a filter and the portfolio items.

    Connecting the filters to the items

    The main thing you probably want to know is how to connect the filter with the items. It's simple:

    • Create a filter and add a data-filter for your category, don't forget to add the . before it, example: 
    <div class="portfolio__filters justify-content-md-center d-flex text-center">
    <a href="#" data-filter="*" class="current">All</a>
    <a href="#" data-filter=".branding">Branding</a>
    <a href="#" data-filter=".digital">Digital</a>
    <a href="#" data-filter=".graphic-design">Graphic Design</a>
    <a href="#" data-filter=".photography">Photography</a>
</div>
    • Next add the item in the portfolio container. Add the categories as classes to the item to connect it to the filter, example:
    <a href="/single-portfolio.html" class="grid-item branding digital isotope-item">
    <div class="portfolio__item">
        <div class="item__image">
            <img src="/assets/img/portfolio/portfolio-1.jpg" alt="image">
        </div>
        <div class="item__content">
            <div class="item__category"><span>Branding</span></div>
            <div class="item__title"><span>Timeless Campaign <i class="lni-arrow-right"></i></span></div>
        </div>
    </div>
</a>

    Changing settings

    You can find the Isotope settings in /assets/js/custom.js. Please edit the settings according to the documentation of Isotope.
