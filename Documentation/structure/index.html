    Basic Structure

    HTML framework

    This template has been built with Bootstrap. Bootstrap is the most popular HTML framework in the world. Bootstrap is very well documented and is easy to master. If you're new to Bootstrap please go to getbootstrap.com before you start to edit the HTML of this template.

    SCSS (advanced)

    All SCSS files are included into this project and can be compiled with the compiler to your liking. You'll need to compile main.scss in order to create a complete CSS. All default styling variables can be found in: abstract/scss/variables.scss. You can generate color-dark.scss and rtl.scss seperatly if you need them. The rest of the structure the SCSS files is straightforward.

    CSS (beginner)

    There are also CSS files available for those that don't want to work with SCSS:

    • main.css - the main CSS file with everything in it
    • color-custom.css - easily change the colors of the template
    • color-dark.css - the dark styled
    • rtl.css - right-to-left stylesheet

    Please make sure that the CSS files are included in the HEAD if you want to use them.

    jQuery

    This template has been built with jQuery. jQuery is a very popular JavaScript library. js/custom.js is the jQuery file for this template. You can change settings and code in this file if needed. Please go to: https://www.w3schools.com/jquery/ if you want to learn jQuery.

    Images

    All images can be found in the images folder. Just open one of the images in your photo editing application and replace the content. This way you'll always use the correct resolution for the images. Good image compression is key for your page speed. Go to https://squoosh.app if you need good compression for your images.

    Fonts

    This template makes use of the Google Font Poppins. You can use any font you want. Just replace the font in the head of all the HTML files and make sure it's set in SCSS and/or CSS.

    Icons

    LineIcons are used inside this template. You can find all of the icons at: https://lineicons.com.

    Animations

    You can simply add or edit appear-animations within the template. Just add data-aos="fade" to your div and add the aos-init class. The animation will play when the div comes in your screen. You can find all possibilities at https://michalsnik.github.io/aos/.

    Minified versions

    There are also concatenated minified versions of both of the JS and the CSS files. These will give you the best pagespeed results if needed, but it's nearly impossible to edit these files. You can however minify edited files yourself, but please note that minifying your files is not supported by the theme author. To use the existing minified files you have to edit the header and footer of your HTML files. Remove the CSS files and replace them with:

        <link rel="stylesheet" href="assets/css/vendors.min.css">
    <link rel="stylesheet" href="assets/css/main.min.css">

    You can still add the RTL or color stylesheets at the bottom of these stylesheets.

    For the JS files do as follow: Remove the JS files and replace it with:

        <script src="assets/js/main.min.js"></script>
